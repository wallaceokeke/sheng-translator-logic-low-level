## 🚀 How It Works  

1. **Speech Recognition** → Google Speech-to-Text / CMU Sphinx  
2. **Translation Logic** → Rule-based NLP, tokenization, slang mapping  
3. **Speech Output** → pyttsx3 / gTTS for natural voice
